# ENGO551_Lab1

import.py is my code to import the books.csv data into a database

application.py contains my flask scripts that is the bridge between the database and the html scripts.

the static folder contains the stylesheets (stylesheet.scss and stylesheet.css) as well as libary.jpeg which is the background image for the website

the templates folder contains the html files.
  layout.html is the backbone of the structure and design, and the other HTML files inherit from it.
  login.html is the login page
  home.html is the home page
  results.html is the page for showing query results
  viewbook.html is the page that shows a book's information with more detail
